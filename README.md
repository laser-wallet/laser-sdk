# Laser Wallet SDK

### SDK to interact with a Laser wallet.

## Usage

```
npm i laser-sdk
```

### Import with TypeScript:
```ts
import { ethers } from "ethers";
import { Laser, LaserFactory} from "laser-sdk";
```

### Import with JavScript:
```js
const { ethers } = require("ethers");
const { Laser } = require("laser-sdk");
```
